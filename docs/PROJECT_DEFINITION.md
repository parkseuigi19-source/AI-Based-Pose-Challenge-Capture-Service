# 프로젝트 정의서 (Project Definition)

## 1. 프로젝트명
**AIX Mini Project (AI 기반 동작 인식 및 분석 웹 서비스)**

## 2. 프로젝트 개요
본 프로젝트는 **YOLOv8**과 **MediaPipe** 등의 최신 컴퓨터 비전 기술을 활용하여 사용자의 동작(Pose)을 실시간으로 인식하고 분석하는 웹 애플리케이션입니다. **FastAPI**를 백엔드로 사용하여 가볍고 빠른 서버 환경을 구축하였으며, 사용자 친화적인 웹 인터페이스를 통해 누구나 쉽게 자신의 동작을 촬영, 분석, 피드백 받을 수 있습니다.

## 3. 기획 의도 및 배경
- **홈 트레이닝 및 헬스케어 수요 증가**: 비대면 운동 코칭 및 자세 교정에 대한 니즈 반영
- **AI 기술의 접근성 확대**: 복잡한 AI 모델을 웹 브라우저를 통해 쉽게 체험할 수 있는 플랫폼 필요
- **동작 데이터 수집 및 시각화**: 자신의 운동 자세를 객관적인 데이터(스켈레톤, 점수)로 확인하고자 하는 요구 충족

## 4. 주요 목표
1. **실시간성 보장**: 웹캠을 통해 지연 없이 동작을 인식하고 시각화
2. **정확한 자세 분석**: YOLOv8 Pose 모델을 활용하여 정밀한 관절 포인트 추출
3. **편의성 제공**: 별도의 설치 프로그램 없이 웹 브라우저에서 실행 가능
4. **결과 피드백**: 직관적인 점수와 이미지 캡처를 통해 사용자에게 성취감 부여

## 5. 기대 효과
- 사용자는 전문적인 장비 없이도 자신의 자세를 스스로 점검할 수 있음
- 운동 자세 교정, 댄스 연습, 재활 치료 보조 등 다양한 분야로 확장 가능
- AI 비전 기술을 활용한 인터랙티브 웹 서비스 개발 경험 축적 및 학습 자료 활용
