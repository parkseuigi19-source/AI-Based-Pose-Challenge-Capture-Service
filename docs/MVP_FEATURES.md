# MVP 기능 정의서 (Minimum Viable Product Features)

## 1. 메인 홈 (Home)
- **기능**: 서비스 소개 및 시작 진입점
- **구성 요소**:
    - 프로젝트 로고 및 타이틀
    - '시작하기' 버튼 (설정 또는 플레이 화면으로 이동)

## 2. 환경 설정 (Settings)
- **기능**: 사용자 맞춤 설정 제공
- **세부 기능**:
    - **난이도 조절**: 초급/중급/고급 등 동작 인식 민감도 또는 목표 설정
    - **소스 선택**: 웹캠 실시간 입력 또는 비디오 파일 업로드 선택
    - **음악/효과음**: 배경음악 표출 여부 설정

## 3. 플레이 모드 (Play Mode - Core Feature)
- **기능**: AI 기반 실시간/비디오 동작 인식 및 가이드
- **세부 기능**:
    - **웹캠 뷰어**: 실시간 사용자 영상 스트리밍
    - **스켈레톤 오버레이**: 인식된 신체 관절(Keypoints)을 영상 위에 시각화 (MediaPipe/YOLO)
    - **캡처 기능**: 특정 동작 순간을 캡처하여 저장 (자동/수동)
    - **실시간 피드백**: 동작의 정확도에 따라 시각적 피드백 제공 (OK/NG 등)

## 4. 결과 리포트 (Results)
- **기능**: 플레이 수행 결과 분석 및 표시
- **세부 기능**:
    - **점수 표시**: 동작 정확도를 기반으로 한 스코어링
    - **캡처 갤러리**: 플레이 중 저장된 주요 동작 이미지 확인
    - **비디오 다시보기**: 업로드하거나 녹화된 영상 다시보기
    - **데이터 시각화**: 날짜별, 폴더별 수행 기록 요약 (이미지 카운트, 최고 점수 등)

## 5. 비디오 업로드 및 분석
- **기능**: 저장된 동영상 파일을 업로드하여 AI 분석 수행
- **세부 기능**:
    - **파일 업로드**: MP4 등 비디오 파일 업로드 인터페이스
    - **서버 처리**: 업로드된 영상을 프레임 단위로 분석하여 결과 데이터 생성
    - **결과 연동**: 분석 완료 후 결과 페이지로 자동 리다이렉트
